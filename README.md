# Home Assistant Add-ons

Personal collection of Home Assistant add-ons.

## 🐳 Docker Socket Proxy

Secure proxy for Docker daemon access with fine-grained API permissions.

- **Port:** 2375
- **Features:** Per-endpoint access control, default-deny security

## 🤖 Obico ML

AI-powered 3D printer failure detection via webcam analysis.

- **Port:** 3333 (REST API)
- **Features:** Real-time detection, pre-trained models, CPU/GPU support
