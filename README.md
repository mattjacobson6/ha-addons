# Home Assistant Add-ons

Personal collection of Home Assistant add-ons.

## 🐳 Docker Socket Proxy

Secure proxy for Docker daemon access with fine-grained API permissions.

- **Port:** 2375
- **Features:** Per-endpoint access control, default-deny security

## 🤖 Obico ML

AI-powered 3D printer failure detection via webcam analysis.

- **Port:** 3333 (REST API)
- **Features:** Real-time detection, pre-trained models, CPU/GPU support

## 📊 cAdvisor

Container Advisor for resource monitoring and performance analysis.

- **Port:** 8080
- **Features:** Live metrics, performance analysis, resource usage tracking

## 📈 Prometheus Node Exporter

Hardware and OS metrics exporter for Prometheus.

- **Port:** 9100
- **Features:** System metrics (CPU, memory, disk, network), Prometheus-compatible output

## 🚀 Hawser Agent for Dockhand

Docker management agent that bridges Home Assistant to Dockhand server.

- **Modes:** Edge (WebSocket connection) or Standard (Docker API port)
- **Ports:** 2376 (Standard mode Docker API)
- **Features:** Container/image/network/volume/stack management, ingress web UI

## 🗄️ pgAdmin4

PostgreSQL database management and administration web interface.

- **Port:** 80/443 (HTTP/HTTPS)
- **Features:** Database management, query tools, PostgreSQL 12-17 support, SSL/TLS
