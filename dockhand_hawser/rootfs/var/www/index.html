Hawser Agent for Dockhand

The Hawser agent is running inside this add-on. There is no interactive UI here — manage this Docker host from your Dockhand dashboard.

  • Check the add-on Log tab for connection status and the resolved Hawser version.
  • Edge mode: the host appears in Dockhand once the agent connects.
  • Standard mode: add the host in Dockhand at http://<home-assistant-ip>:__STANDARD_PORT__.

Full documentation is on the add-on's Documentation tab.
